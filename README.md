# 🍽️ Zomato Dashboard using Streamlit

An interactive dashboard to analyze and visualize Zomato restaurant data using Streamlit, Pandas, Plotly, and Seaborn.

🔗 **Live App**: [Click here to launch the dashboard](https://zomato-dashboard-app-ostmvkjsyvvtuavpvzpj8k.streamlit.app/)

---

## 📊 Features

- 📍 Top 10 restaurant locations
- 🍽️ Cuisine distribution pie chart
- ⭐ Average rating with filters
- 💰 Cost vs Rating scatter plot
- 🔍 Filter by location and cuisine
